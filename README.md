# No Detalhe

Site institucional e guia de servicos para estetica automotiva.

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion

## Como rodar

```sh
npm install
npm run dev
```

## Scripts

- `npm run dev`: inicia ambiente de desenvolvimento
- `npm run build`: gera build de producao
- `npm run test`: executa testes unitarios
- `npm run lint`: valida regras de lint

## Estrutura principal

- `src/features/home`: modulo da Home alinhado ao SDD
- `src/components/ui`: componentes base do design system
- `public`: arquivos estaticos (robots e sitemap)
